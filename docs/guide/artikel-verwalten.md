# Artikel verwalten

## Neuen Artikel erstellen

1. Klicken Sie auf das **+** Symbol neben der Ueberschrift "Knowledge Base" in der Sidebar.
2. Geben Sie einen **Titel** ein.
3. Waehlen Sie optional eine oder mehrere **Kategorien** aus.
4. Verfassen Sie den Inhalt im **Rich-Text-Editor**.
5. Setzen Sie den **Published**-Schalter, wenn der Artikel sichtbar sein soll.
6. Klicken Sie auf **Save**.

Um einen Artikel direkt in einer bestimmten Kategorie zu erstellen, klicken Sie auf das **+** Symbol neben dem Kategorienamen.

## Artikel bearbeiten

1. Waehlen Sie den Artikel in der Sidebar aus.
2. Klicken Sie auf **Edit** in der Kopfzeile des Artikels.
3. Nehmen Sie die gewuenschten Aenderungen vor.
4. Tragen Sie eine kurze **Change Summary** ein, die beschreibt was geaendert wurde.
5. Klicken Sie auf **Save**.

Jede Speicherung erstellt automatisch eine neue Version des Artikels.

## Artikel loeschen

1. Waehlen Sie den Artikel in der Sidebar aus.
2. Klicken Sie auf **Delete** in der Kopfzeile.
3. Bestaetigen Sie die Loeschung im Dialog.
