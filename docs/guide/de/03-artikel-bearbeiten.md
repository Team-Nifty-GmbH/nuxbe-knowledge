# Artikel bearbeiten

1. Wählen Sie den gewünschten Artikel in der Sidebar aus.

2. Klicken Sie auf **Bearbeiten**.

   ![Bearbeiten-Button hervorgehoben](../screenshots/07-bearbeiten-button.png)

3. Nehmen Sie Ihre Änderungen vor. Beim Bearbeiten eines bestehenden Artikels erscheint zusätzlich das Feld **Zusammenfassung**. Tragen Sie hier ein, was Sie geändert haben. Diese Information wird in der Versionshistorie gespeichert.

   ![Bearbeitungsmodus mit Zusammenfassungsfeld](../screenshots/08-artikel-bearbeiten.png)

4. Klicken Sie auf **Speichern**, um die Änderungen zu übernehmen, oder auf **Abbrechen**, um den Bearbeitungsmodus zu verlassen.

## Artikel löschen

1. Wählen Sie den Artikel in der Sidebar aus.
2. Klicken Sie auf **Löschen** (rot).
3. Bestätigen Sie den Vorgang im Bestätigungsdialog.
