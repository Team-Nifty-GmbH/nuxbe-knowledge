# Versionshistorie

Jede Aenderung an einem Artikel wird als neue Version gespeichert. So laesst sich nachvollziehen, wer wann welche Aenderungen vorgenommen hat.

## Versionen anzeigen

1. Waehlen Sie einen Artikel in der Sidebar aus.
2. Klicken Sie auf **Version History** in der Kopfzeile.
3. Es oeffnet sich ein Dialog mit allen Versionen des Artikels, sortiert nach Versionsnummer (neueste zuerst).

Jede Version zeigt:

- **Versionsnummer** (v1, v2, ...)
- **Titel** zum Zeitpunkt der Version
- **Change Summary** (wenn beim Speichern angegeben)
- **Erstellungsdatum**

## Versionen vergleichen

1. Oeffnen Sie die Versionshistorie eines Artikels.
2. Klicken Sie bei der ersten Version auf **A**.
3. Klicken Sie bei der zweiten Version auf **B**.
4. Klicken Sie auf **Compare Versions**.

Es wird ein Side-by-Side-Diff angezeigt, der die Unterschiede zwischen den beiden Versionen auf Wortebene hervorhebt.
