# Kategorien

Artikel koennen einer oder mehreren Kategorien zugeordnet werden. Kategorien werden ueber das Flux-Core Kategoriesystem verwaltet.

## Kategorien zuweisen

Beim Erstellen oder Bearbeiten eines Artikels koennen Sie im Feld **Categories** eine oder mehrere Kategorien auswaehlen. Das Feld bietet eine Suchfunktion und laedt verfuegbare Kategorien automatisch.

## Kategorien in der Sidebar

Die Sidebar zeigt Kategorien als aufklappbare Baumstruktur:

- **Hauptkategorien** werden mit einem Pfeil-Symbol angezeigt. Klicken Sie darauf, um die zugehoerigen Artikel anzuzeigen.
- **Unterkategorien** erscheinen eingerueckt unterhalb ihrer Hauptkategorie.
- **Unkategorisierte Artikel** werden unterhalb der Kategorien angezeigt.

## Kategorien verwalten

Kategorien werden zentral im Flux-ERP Kategoriesystem verwaltet. Dort koennen Sie neue Kategorien anlegen, umbenennen oder die Sortierreihenfolge aendern.
